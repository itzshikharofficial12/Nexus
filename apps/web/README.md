# DailyOS Web

Next.js 14 frontend application with TypeScript, Tailwind CSS, and App Router.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Project Structure

- `app/` - App Router pages and layouts
- `components/` - Reusable React components
- `features/` - Feature-specific components and logic
- `lib/` - Utility functions and helpers
- `store/` - State management (if using)

## Dark Mode

Dark theme support is built-in with CSS variables. Use Tailwind's `dark:` prefix for dark mode styles.
